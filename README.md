# Berbersite

Berber salonu icin tanitim odakli web arayuzu. React (CRA) uzerinde TypeScript ve Tailwind ile gelistirilmistir.

## Teknolojiler
- React 19 + Create React App
- TypeScript
- TailwindCSS
- React Router
- Axios

## Kurulum
```bash
npm install
```

## Gelistirme
```bash
npm start
```
Uygulama `http://localhost:3000` adresinde acilir.

## Build
```bash
npm run build
```

## Test
```bash
npm test
```
